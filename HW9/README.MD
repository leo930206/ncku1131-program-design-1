功能說明

1. 基本運算函數：
   - power(double x, int n)：計算 x^n，其中 x 為浮點數，n 為整數。
   - multiply(double x, int n)：計算 x * n。
   - divide(double x, int n)：計算 x / n，並檢查 n ≠ 0 以避免除以零導致錯誤。

2. 複合運算函數：
   - powerpower(F func, double x, int n, int m)：計算 (func(x, n))^m，適用於次方、乘法或除法結果的多次方運算。
   - powermultiply(F func, double x, int n, int m)：計算 func(x, n) * m，適用於次方、乘法或除法結果的乘法運算。
   - powerdivide(F func, double x, int n, int m)：計算 func(x, n) / m，適用於次方、乘法或除法結果的除法運算。

3. 函數指標：
   使用 typedef 定義函數指標類型 F，以簡化函數指標的使用，允許將 power、multiply 或 divide 函數動態傳遞至複合運算函數。

4. 命令列參數：
   - 程式透過命令列參數取浮點數 x、整數 n 與 m。
   - 使用輸入的值計算符合運算邏輯，並依序輸出 9 種結果。

5. 輸出結果：
   每行輸出一種運算結果，包含：
   - (x^n)^m
   - (x*n)^m
   - (x/n)^m
   - (x^n)*m
   - (x*n)*m
   - (x/n)*m
   - (x^n)/m
   - (x*n)/m
   - (x/n)/m

設計細節

1. 精度與溢出處理：
   - 使用 double 型別進行運算，並加入溢出檢查，若結果超出 DBL_MAX，程式會提示錯誤並中止。
   - 在 divide 函數中，避免整數除法問題，確保分母轉換為浮點數運算。

2. 錯誤檢查：
   - 檢查命令列參數數量是否正確，若不符要求則提示用法並中止程式。
   - 若 n 或 m 為負數，程式將提示錯誤並中止。
   - 避免分母或除數為 0 的情況，並給出錯誤提示。

3. 程式結構：
   - 基本運算與複合運算分開實現，透過函數指標動態選擇運算方式。
   - 將所有輸出統一格式化為小數點後 6 位，方便檢視結果。

4. 靈活性：
   - 程式設計簡潔，可輕鬆擴展新的運算函數或複合運算。
